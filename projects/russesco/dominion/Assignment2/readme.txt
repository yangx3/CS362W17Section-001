How to Run the Random Tester code on Unix System:
1. Copy All Files and Folders over. 
2. cd your way into this folder (Where this readme.txt and the pom.xml is)
3. mvn compile
^^^^^^^ Compiles maven
4. mvn test
^^^^^^^^ Runs the Unit tests for Dominion
5. Code should exit with 0 failures, 0 erros and 0 skips.

6. You can look at all of the Evosuite Test Cases inside the src/test folder
7. There is also a code coverage html report for the Evosuite Testing in this folder.
