Compile -> Run without code coverage: Must be in the same directory as the pom.xml

1. Type the following

mvn compile

2. Then type

mvn package

3. Then type

mvn test



Compile -> with code coverage: Must be in the same directory as the pom.xml

1. Type the following

mvn compile

2. Then type

mvn package

3. Then type

mvn cobertura:cobertura
