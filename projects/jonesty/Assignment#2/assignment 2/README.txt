
To Compile: run 'mvn compile'. This compiles all of the files in src/main/java/org/cs362/dominion. All of my Dominion code is contained in these files.
To run Tests: run 'mvn test'
to generate html report: run 'mvn cobertura:cobertura'
to simulate a whole game of Dominion (RandomTestDominion), compile all of the files in scr/main/java/org/cs362/dominion, and then run RandomTestDominion.
ALL my tests generated by EvoSuite exist in src/test/java/real_dominion
