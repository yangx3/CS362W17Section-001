To compile:
	mvn compile
	
Package:
	mvn package
	
To run the tests:
	mvn test
	
To run the game:
	java -jar ./target/dominion_v2-0.0.1-SNAPSHOT.jar <#OfKingdomCards 1-10>

The coverage for the assignment is in an html on the page


To clean:

	mvn clean
	
This may be required to run and test correctly
