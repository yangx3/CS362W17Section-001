
Run 'make clean' to remove unnecessary files

Run 'make compile' to compile the project

Run 'make build' to compile and run all tests and create the jar
 
Run 'make test' to run all of the tests (Evosuite and random tester I wrote)

Run 'make coverage' to run the tests while calculating code coverage (which are written to target/site/cobertura)

Run 'make run2bot' to have two bots play each other in a full game of Dominion (make sure to run 'make build' first)

Run 'make runRandTester' to execute the only the random tester with a seed of 10
