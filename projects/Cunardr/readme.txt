Download dominion folder, run eclipse with codecover and junit 
available for use. In eclipse, open project and select the dominion folder.
For main program, open driver.java (found in dominion/src/dominion).
To run the program in full (without testing) uncomment the line "Play.playGame(0);"
and comment out the function calls to pt.
To run the test setup, uncomment out the calls to pt, comment out play game.
In the package explorer, right click on PLayerTest1.java, select run as... 1 Junit test
To run coverage test, right click all files and select use for coverage measurement
Right click driver.java, select run as... runconfigurations, select the codeCover 
tab, and check 'Run with CodeCover"
