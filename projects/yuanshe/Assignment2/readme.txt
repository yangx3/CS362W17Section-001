

a. Dominion —— the maven project folder.

b. Dominion/randoop-all-3.0.9.jar —— the Randoop tool that generate random test suites.

c. generated-test-files —— randomly generated Junit code by Randoop

d. coverage-report —— html report generated by Cobertura for above generated test suites

e. RandomTestDominion.java —— Random Tester


How to Run: (MAVEN-3.3.9 and Randoop-3.0.9 are used)

1. Generate Junit test files: In the directory Dominion (where pom.xml is put)
java -cp randoop-all-3.0.9.jar:target/classes randoop.main.Main gentests --classlist=testclasses.txtput generated Junit test codes to src/test/java/org.cs362.dominion2. Compile: 
mvn compile

3. Build
mvn package

4. Run Junit test cases
man test

5. Generate code coverage report
man cobertura:cobertura
